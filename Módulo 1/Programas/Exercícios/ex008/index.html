Principais formatações

Negrito / Destaque:

Nesta frase, temos um termo em negrito usando a tag <b> (não semântico).

Nesta frase, temos um termo em destaque usando a tag <strong> (semântico).

Itálico / Ênfase:

Nesta frase, temos um termo em itálico usando a tag <i> (não semântico).

Nesta frase, temos um termo em ênfase usando a tag <em>; (semântico).

Texto marcado:

Podemos criar, também, um texto marcado usando a tag <mark>

E no outro parágrafo temos, também, outro texto marcado no final.

Texto grande e pequeno:

Estamos criando um texto grande e texto pequeno nesse parágrafo, usando, respectivamente, as tag <big> (que está sendo caracterizada no VSC como obsoleta) e <small>.

Texto deletado:

Podemos marcar um texto deletado para indicar que ele deve ser lido mas não considerado, usando a tag <del>.

Texto inserido:

Podemos marcar um texto inserido para dar ênfase e indicar que ele foi adicionado posteriormente usando a tag <ins>.

Existe, também, o sublinhado com a tag <u>; (não semântico).

Texto sobrescrito:

Para iserir coisas do tipo: x2+5x+6. Para isso, basta utilizar a tag <sup>

Texto subscrito:

Para inserir coisas do tipo: H2O. Para isso, basta utilizar a tag <sub>.
